LD_PRELOAD 预加载绕过

每个程序执行的时候会去动态链接库so文件里面找函数的位置，而我们的目的是让程序去执行我们自定义的动态链接库

LD_PRELOAD这个全局变量指定的so文件会在每个程序本身的so文件之前加载

有sendmail

查看程序执行了哪些函数。

比如 id中执行了 getuid()

所以可以编写出 test.c。

利用

在php中如果把so文件传上去了后，可以利用putenv函数来设置全局变量
